# TikTok Quality Method

A Windows batch workflow using FFmpeg to modify timing metadata for smoother/better looking TikTok uploads.

---

## Overview

This method processes a video using FFmpeg timing flags.

The resulting file may appear slowed, broken, or incorrect when played locally.
This is expected behavior.

After upload, TikTok interprets the timing metadata differently, resulting in smoother playback and reduced compression artifacts.

---

## What This Is

• A metadata-based timing workflow
• A Windows `.bat` script using FFmpeg
• Designed for desktop browser uploads
• Intended for creators working with 60–120 FPS footage

---

## What This Is Not

• Not a resolution upscaler
• Not a bitrate booster
• Not guaranteed to bypass all platform compression

---

## Requirements

• FFmpeg (placed in the same folder as the script)
• Desktop browser upload
